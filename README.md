# Sistema de Agendamento Médico

Projeto simples em Python para simular agendamento de consultas médicas pelo terminal.

## Funcionalidades

- Escolha de especialidades médicas  
- Agendamento de horários disponíveis  
- Remoção do horário agendado  
- Triagem para quem não sabe qual médico procurar  

## Tecnologias usadas

- Python  

## Objetivo


- Praticar lógica de programação e estruturas básicas do Python.
